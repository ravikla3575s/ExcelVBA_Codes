医薬品名変換ツール Mac版インストール手順
======================================

1. インストール

このツールは特別なインストール作業は必要ありません。
フォルダごと任意の場所（書類フォルダなど）にコピーして使用できます。

2. 初期設定

初回起動時には以下の設定を行ってください。

a) マクロの有効化

1. Excelで「棚名一括設定マクロv5.0.0.xlsm」を開いたときに
   セキュリティ警告が表示される場合は「コンテンツを有効にする」をクリック
2. または、Excel の環境設定からマクロセキュリティを調整してください
   (Excel → 環境設定 → セキュリティ → マクロセキュリティ)

3. Mac版の特記事項

Mac版では、Windows版で使用されているMicrosoft Forms 2.0 Object Libraryが
利用できないため、進捗表示の機能を修正して、代わりにExcelのステータスバー
（画面下部）を使用するようにしています。

4. ソースコードについて

src フォルダに含まれる .bas ファイルは、必要に応じて VBA エディタから参照できます。
既存マクロに機能を追加したい場合は、これらのモジュールをインポートしてください。

VBA エディタでのモジュールのインポート方法:
1. Option+F11 キーを押して VBA エディタを開く
2. 「ファイル」→「ファイルのインポート」を選択
3. src フォルダ内の .bas ファイルを選択してインポート

5. トラブルシューティング

- マクロが動作しない場合: マクロが有効になっているか確認してください
- パフォーマンスが悪い場合: Mac版Excelでは一般的にVBAの実行速度がWindows版より遅い傾向があります
- ファイルが見つからないエラーが発生する場合: フォルダ構造が維持されているか確認してください

6. 動作環境

- macOS 10.14以降
- Microsoft Excel for Mac 2016以降
- VBAマクロが有効化されたExcel環境
